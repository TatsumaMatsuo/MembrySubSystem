# 画面モックアップ

要件定義書 §7.2「最初に画面イメージ(モックアップ)で全体の使い勝手をすり合わせたい」に対応する、**すり合わせ用の使い捨てプロトタイプ**。

> ⚠️ **数値はすべてダミー**。依存なしの単一HTML(インラインSVG/CSS)で、ブラウザにドラッグ&ドロップするだけで開けます。本番は recharts + Lark Base 実データで実装します。

## `keiei-dashboard.html` の見どころ

1. **中計KGI 3枚カード**(売上67億 / ROA13% / 労働生産性10百万円/人)
   - 各カードに **50→51→52期の目標トラジェクトリ**(紺の破線=線形補間)と **50期の年換算実績**(赤点)を折れ線表示。
   - 「最終52期目標への到達度」を一目で把握。
2. **全社KPI(年度PL)**: 売上/粗利/営業利益/製造原価率/外注発注率/販管費 の 年度目標 vs 実績累計 を進捗バー+緑黄赤判定で表示。
3. **会計データ入力状況**: 科目ごとの入力粒度(月別/四半期/半期)と確定状況。「本部長が一括入力」の運用を可視化。
4. 生産本部KPI(Lv2)への相互リンク(同一の会計データを参照する旨を明示)。

## `seisan-kpi-dashboard.html` の見どころ

Excel「01_ダッシュボード」を Web 化したもの。**会議投影での月次レビュー**を想定。

1. **月次レビュー進行ガイド**を最上部に常設(①信号盤→②赤黄件数→③KPI特定→④施策決定→⑤★確認)。
2. **経営KPI信号盤(Lv2 7指標)**: 粗利率/総資産回転率/材料金額比率/クレーム/社内不具合/在庫+仕掛/外注金額 を 値+目標+緑黄赤。左帯の色で一目判別。
3. **★達成ランキング**: 製造部6課を★数で降順表示(北多久縫製28がトップ)+ 生産管理部3課。
4. **要対応KPI**: 赤18件/黄14件 + 主要な未達KPIリスト。
5. **施策進捗**: 継続/強化/見直し/完了・改善/悪化 の件数。
6. **月次トレンド折れ線**: 月次粗利率(目標35%) と 在庫+仕掛(目標447百万円) を8ヶ月推移で表示。
7. Lv2経営KPIは「経営」エリアの会計データ(`KAIKEI_ACTUAL`)を参照する旨を明示。

## `seisan-kpi-input.html` の見どころ

Excel「02_KPI実績入力」を Web 化。**入力即判定**が主眼。

1. **行=KPI、列=固定情報＋12ヶ月＋年累計/進捗率/判定**。本社鉄工課の6KPIを実値で表示。
2. **セルの状態を色で区別**: 確定9ヶ月=グレー(ロック) / 5月=黄(入力対象) / 6-7月=未来 / 右3列=自動計算(グレー)。
3. **タブ**: 通常KPI / 基礎データ算出KPI。
4. **重要な確定事項の反映**: Lv2算出KPI(粗利率等)は当画面で入力せず、「経営」エリアの会計データを参照する旨を緑のインフォボックスで明示。

## `seisan-kpi-measures.html` の見どころ

Excel「03_施策管理」を Web 化。**グループ単位**のPDCA(マスター・ディテール型)。

1. **グループ単位の表示**(M:N): グループセレクタで切替。**北関東鉄工課は鉄工課Gと北関東工場Gに重複所属**(⮂表示)。
2. **施策:PDCA = 1:多**: 左=施策一覧(件数無制限・`＋追加`)、右=選択施策のPDCA全月。行クリックで切替。
3. **施策の状態**(実施中/完了/中止/下書き)・**基準値→現在値・狙い値**を表示。
4. **月次PDCAログ**: 計画/実施/対象KPI実績/効果/本部長コメント/翌月アクション。効果は対象KPI実績の変化から**自動判定→責任者確定**。
5. **過去施策の参照**: 期またぎで同一KPIへの打ち手と効果を時系列表示。
6. 行レベル権限(計画/実施/効果=責任者、本部長コメント/翌月アクション=本部長)を注記。

> 操作: グループ切替 / 施策行クリック / ＋施策追加 が実際に動きます。

## `seisan-kpi-stars.html` の見どころ

Excel「04・05 ★達成表」を Web 化。**★達成評価は部署ごと**(Lark部署単位)。

1. **★達成ランキング**: 部署の総合計★を降順(🥇🥈🥉)。タブで製造部6課/間接3課を切替。
2. **部署別★表**: 行=項目(品質クレーム/不具合/納期LT/原価生産効率 等)×列=8月〜7月。月間目標達成で★(自動)。
3. **合計★数(自動)** + **5S大賞/労災(手入力・黄セル)** + **総合計★数**。
4. 数値は Excel 04・05 の実値(北多久縫製28がトップ等)。
5. 集計は**部署単位**(グループ集約ではない)。部署=Lark部門参照。

## `seisan-kpi-master.html` の見どころ(管理者専用)

1. **KPIマスタ管理**: 63指標の一覧・編集(名称/単位/集計/方向/目標/オーナー/有効)。フィルタ。**期切替=新期作成(前期複製)**。部署は Lark Contact 部門から選択。
2. **グループマスタ管理**: グループ一覧 + **所属マトリクス(行=Lark部署 × 列=グループ)**。チェックで所属設定。**北関東鉄工課が鉄工課G・北関東工場Gの両方にチェック**(⮂重複)= M:Nを直感的に設定。

## `seisan-kpi-history.html` の見どころ

1. **3スコープのタブ抽出**: **全社・部門 / 部署別 / グループ別**。
   - 全社・部門: 43-49期推移 + 50期目標を折れ線(青破線=目標)+表で表示。
   - 部署別: 部署(Lark部署)を選択 → KPI別に 49期実績/50期目標/50期現在/判定。
   - グループ別: グループ選択 → 所属部署を合算(累計系)/平均(率系)。**北関東鉄工課は複数Gで重複計上**(注記表示)。
2. **目標妥当性の判定**(ストレッチ/妥当/要努力)を併記。例: クレーム目標6=過去最少10を下回る高ストレッチ、鉄工一人当たり4.1=過去平均圏で妥当。
3. データ源: 50期〜=当システム実績(部署別が充実)、43-49期=移行レガシー(集約中心)。
4. 操作: スコープタブ / 部署セレクタ / グループセレクタ が動きます。

## レビューでの確認ポイント(たたき台)

- KGIカードのトラジェクトリ表現は会議投影で分かりやすいか。
- 全社KPIに出す勘定科目はこの6つで十分か(過不足)。
- 会計データ入力の粒度表示・確定状況の見せ方は運用に合うか。
- 「経営」エリアと「生産本部KPI」エリアの行き来の動線。

## 開き方

エクスプローラで `keiei-dashboard.html` をダブルクリック、またはブラウザにドラッグ&ドロップ。
会議投影時はブラウザを全画面(F11)推奨。
