# 参考図台帳検索 — 汎用マスタ（候補ソース）メモ

検索画面の★項目の候補は2つのマスタから供給する。

## 1. 部材記号17項目 → 参考図面部品マスタ

`C1 / T1 / G1 / B1〜B4 / P1〜P4 / Ga / Gc / WB / ST / 柱ラチ / 梁ラチ`
→ 参考図面部品マスタ（`分類1`＝記号 → `部品名称`＝候補）。台帳の該当列に**完全一致**で絞り込み。

## 2. それ以外の★ → 汎用マスタ（全社共通・`tbl9MGRMMHGNXFUU`）

全社共通マスタを使用。**システム名＝「参考図面情報」固定**で、**項目名**を抽出条件に候補（`内容`）を取得する。

- テーブル構造: `システム名 / 項目名 / 内容 / 備考1 / 備考2 / 備考3`
- 参考図で使う項目名（24種・例）: `用途` `柱形状` `壁面` `土間` `基礎形状` `BPL` `出入口種類`
  `F1布基礎`/`F1独立基礎`/`F1H鋼材`（F2/F3/FGも布基礎・独立基礎別）
  `形状関連` `出入口関連` `膜関連` `設備関連` `構造関連` `移動建屋関連` `開閉関連` `畜舎関連`

### 列 → 項目名 の対応（page.tsx `MASTER_ITEMS`）

| 検索列 | 汎用マスタ 項目名 | 照合 |
|---|---|---|
| 用途 / 柱形状 / 壁面 / 土間 / 基礎形状 | 同名 | 完全一致 |
| B-PL形状 | `BPL` | 完全一致 |
| 出入口1 / 出入口2 | `出入口種類` | 完全一致 |
| F1 | `F1布基礎`+`F1独立基礎`+`F1H鋼材`（統合） | 完全一致 |
| F2 / F3 / FG | `F2…`/`F3…`/`FG…`（布基礎・独立基礎を統合） | 完全一致 |
| 形状/出入口/膜/設備/構造/移動建屋/開閉/畜舎 関連 | 同名 | **部分一致**（台帳の関連列は複数キーワードの複合文字列のため） |

### 汎用マスタに無い★（台帳実値をフォールバック表示）

`建屋区分` `申請有無` `設計ルート` `柱成` `梁成` は汎用マスタに項目が無いため、台帳の実値から候補生成。
`サイズ1` `サイズ2` は候補が数百件のため部分一致テキスト入力。

> 候補の追加・修正は汎用マスタ（Lark）側で運用。アプリは読み取りのみ（システム名=参考図面情報）。
> 実装: 読込=`app/api/eigyo/sankou-zu/route.ts`、対応づけ=`app/eigyo/sankou-zu/page.tsx`。
